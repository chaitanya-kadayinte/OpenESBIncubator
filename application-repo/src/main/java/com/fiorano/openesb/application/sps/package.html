The com.fiorano.openesb.application.sps package defines a set of DmiObjects that represent a Tifosi Service. This package defines ServicePropertySheet which represents a Tifosi Service; and also defines other classes that contain information about the service - PortDescriptor, Resource, Execution, and Deployment, for example.

This package is used in transferring service information from the Tifosi Enterprise Server to a Tifosi Peer Server and from the Tifosi Enterprise Server to a Tifosi Tool. It is also used in saving a new or modified service in the Tifosi Enterprise Server's Service Repository as an XML file.

Note: For transferring the service information over the Messaging Bus Server, the objects are converted to a byte array as the Messaging Bus expects data in byte array format. At the target server, the byte array is converted back to the appropriate DMI object.
